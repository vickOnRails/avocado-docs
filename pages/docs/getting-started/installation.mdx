# Installation

## Add to your project

You can add `avocado-ui` by doing a simple npm/yarn install

- with `npm`

```bash
npm install @avocado-ui/react
```

- or with `yarn`

```bash
yarn add @avocado-ui/react
```

`avocado-ui` should be available to use in your react application. Jump on to the next section for the getting started guide
